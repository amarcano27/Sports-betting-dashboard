## üìù Customization

### Update Personal Information
1. Edit `index.html`:
   - Replace `your.email@example.com` with your email
   - Replace `https://github.com/yourusername` with your GitHub profile
   - Replace `https://linkedin.com/in/yourprofile` with your LinkedIn
   - Update the GitHub repository link for the project

### Update Project Links
- Change `http://localhost:8501` to your deployed Streamlit app URL (if hosted)
- Update the GitHub repository link in the project links section

### Add Screenshots
Replace the placeholder image section with actual screenshots:
1. Take screenshots of your dashboard
2. Create an `images` folder
3. Add screenshots
4. Update the `image-placeholder` div with an `<img>` tag

### Customize Colors
Edit `styles.css` and modify the CSS variables in `:root`:
```css
:root {
    --primary-color: #00E5FF;      /* Your primary color */
    --secondary-color: #8A2BE2;    /* Your secondary color */
    /* ... */
}
```
